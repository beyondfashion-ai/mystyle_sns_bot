# mystyleKPOP Editorial Strategy

## Main: K-POP 패션을 에디토리얼로

### Phase 1 -- 패션 분석 산문 (Fashion Documentary Prose)

팬의 감정이 아니라,
**아티스트의 스타일이 어떻게 설계되었는지**를 해부한다.

#### 1. 강한 오프닝 구조

숫자, 디테일, 구조로 시작한다. 감정으로 시작하지 않는다.

* 예시
  "이번 컴백에서 3가지 다른 실루엣이 등장했다. 모두 같은 컬러 팔레트를 공유한다."

첫 문장은 스타일의 '선택'을 보여준다.
평가하지 않는다. 관찰을 제시한다.

---

#### 2. 구조 중심 서술 -- 감정 대신 분석

* "스타일이 좋다"라고 쓰지 않는다.
  -> **왜 그 스타일이 작동하는지**를 보여준다.
* 패션 요소를 인용하되, 반드시 **시각적 근거와 연결**한다.
* 추상어를 쓰기 전에 구체적 요소로 풀어쓴다.

예시 변환:

* "패션 센스가 뛰어나다"
  -> "오버사이즈 블레이저 위에 벨트로 웨이스트라인을 잡아 구조적 실루엣을 만들었다."

* "스타일이 독특하다"
  -> "블랙 베이스에 네온 그린 한 가지만 포인트로 사용했다."

감정을 설명하지 않는다.
**아이템 -> 조합 -> 효과**의 순서를 기록한다.

---

#### 3. 구체적인 패션 용어

추상 명사 대신:

* 소재명 (시스루, 트위드, 새틴)
* 실루엣 (오버사이즈, A라인, 슬림핏)
* 컬러 (버건디, 차콜, 에메랄드)
* 레이어링 구조
* 액세서리 디테일

"화려한 무대의상" 대신
-> "메탈릭 실버 크롭 재킷에 하이웨이스트 와이드 팬츠를 매치했다."

패션 분석의 현장은
컬러 팔레트, 소재 질감, 실루엣 비율, 스타일링 레이어다.

---

#### 4. 속도감 있는 문장

* 문단은 2~3문장.
* 접속사 최소화.
* 하나의 문단에 하나의 스타일 포인트.

mystyleKPOP의 문장은 짧고 단단하다.
형용사는 구체적이다.
서술은 시각적이다.

---

#### 5. 분석 구조 (일관된 프레임워크)

모든 에디토리얼 콘텐츠는 동일한 분석 구조를 따른다:

1. 컬러 팔레트 분석
2. 핵심 아이템 & 실루엣
3. 레이어링 & 스타일링 구조
4. 액세서리 & 디테일
5. 무대 vs 일상 적용 포인트
6. AI 재해석 인사이트

이 반복이 브랜드 신뢰를 만든다.

---

## 오프닝 패턴 라이브러리

강한 오프닝의 유형:

**Type A: 숫자 오프닝**
"이번 무대에서 {artist}는 4개의 룩 체인지를 보여줬다."

**Type B: 대비 오프닝**
"{artist}의 공항패션은 무대 위와 정반대다. 무대에선 글리터, 공항에선 미니멀."

**Type C: 선택 오프닝**
"블랙 대신 화이트를 선택했다. 이 컬러 전환이 전체 컨셉을 바꿨다."

**Type D: 디테일 오프닝**
"재킷의 숄더 라인이 2cm 더 넓다. 이것만으로 실루엣이 완전히 달라진다."

---

## 언어의 경제성

K-POP 패션 콘텐츠는 단어를 과시하지 않는다.

* "~인 것 같다" 삭제
* "~하게 된다" 삭제
* '의' 남용 최소화
* 중복 표현 제거

예:
* "패션의 스타일의 변화" -> "패션 변화"
* "트렌드를 선도하게 되었다" -> "트렌드를 선도했다"

문장이 날카로워진다.
콘텐츠 신뢰가 올라간다.

---

## 에디토리얼 클로저 (Editorial Closure)

도입부의 키워드를 마지막에 회수한다.

예:

서두:
"이번 컴백에서 3가지 다른 실루엣이 등장했다."

결말:
"3가지 실루엣이 하나의 컨셉 아래 통일되는 순간, 그것이 시그니처 스타일이 된다."

정보로 끝내지 않는다.
**스타일 인사이트**를 남긴다.

## 플랫폼별 적용 가이드

### X (Twitter)
* 280자 제한 안에서 강한 오프닝 + 핵심 분석 1개 + CTA
* 스레드: 오프닝 -> 분석 포인트 3개 -> 클로저 + CTA

### Instagram
* 캡션: 오프닝 -> 분석 구조 전개 -> 클로저 -> CTA
* 카드뉴스: 매거진 문법 (커버 -> 분석 슬라이드 -> 아웃트로)

### Tone 차이
* X: 더 간결, 더 직접적, 팬 참여 유도
* IG: 더 비주얼, 더 서사적, 매거진 무드
