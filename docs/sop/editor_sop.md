## 에디토리얼 레퍼런스: Hypebeast & 데패뉴

mystyleKPOP의 글쓰기 톤은 Hypebeast의 에디토리얼 스타일을 참조한다.

### Hypebeast 핵심 글쓰기 원칙
- **세련되고 개념적인 언어**: 단순 나열이 아닌, 문화적 맥락과 패션 디테일을 연결
- **K-POP을 산업적 관점으로**: 팬덤이 아닌 "fashion's biggest darlings"으로 포지셔닝
- **Bridging 문장 구조**: "~와 ~를 잇는", "~를 재정의하는" 같은 동사 활용
- **클릭베이트 지양**: 세심하게 큐레이션된 캡션, 과장 표현 금지
- **문화를 형성하는 톤**: 단순 리포팅이 아닌 트렌드를 해석하고 방향을 제시

### Hypebeast 헤드라인 패턴
- **트렌드 분석형**: "Is [Category] Entering a New Era?"
- **회고형**: "[X]의 [기간], From [Origin] to [Current]"
- **브랜드 포커스형**: "[Brand] is Leading [Movement/Category]"
- **셀럽 협업형**: "How [Artist] Became [Industry]'s [Superlative]"

### 데패뉴 (Daily Fashion News) 참조
- 간결하고 빠른 패션 뉴스 전달 스타일
- 패션/리테일/뷰티 업계 종사자 대상 전문 정보
- 핵심 요점만 추출하는 편집 능력

### 한국 패션 매거진 레퍼런스

**DAZED Korea:**

## 작업 플로우

### Step 1: 주제 선정
- 아티스트 + 카테고리 조합을 받는다
- 카테고리: comeback_lookbook, style_editorial, vibe_alike, stage_look, weekly_trend, airport_fashion, mv_analysis

### Step 2: 오프닝 작성 (Hypebeast 스타일)
- 에디토리얼 전략의 4가지 오프닝 패턴 중 선택
- 숫자, 대비, 선택, 디테일 중 하나로 시작
- 감정적 표현 금지 (좋다, 멋지다, 대박 등)
- Hypebeast식 문화적 맥락 연결 권장

예시:
- 나쁜 오프닝: "NewJeans의 스타일이 정말 대박이에요! 🔥"
- 좋은 오프닝: "NewJeans가 3번째 컴백에서 선택한 컬러는 단 2가지다."
- Hypebeast식: "Y2K 감성과 미래적 실루엣을 잇는 NewJeans의 최신 룩."

### Step 3: 본문 분석
- 분석 구조를 따름:
  1. 컬러 팔레트
  2. 핵심 아이템 & 실루엣
  3. 레이어링 구조
  4. 액세서리 디테일
  5. 적용 포인트
- 각 포인트는 2~3문장으로 구성
- 구체적 패션 용어 사용 (추상어 금지)
- Hypebeast식 "bridging" 표현으로 포인트 간 연결

### Step 4: 클로저 작성
- 도입부 키워드를 회수
- 스타일 인사이트로 마무리
- 정보 나열로 끝내지 않음
- 트렌드의 방향성을 제시하는 문장으로 마감

### Step 5: CTA 삽입
- my-style.ai 유도를 자연스럽게 배치
- Hypebeast식: "Explore the full editorial at my-style.ai"
- 또는 콘텐츠 맥락에 맞는 한국어 CTA

### Step 6: 안전장치 체크
- [ ] Vibe-Alike 명시 확인
- [ ] 아티스트 간 비교 없음
- [ ] 비하/부정적 표현 없음
- [ ] Bot Disclosure 포함
- [ ] 해시태그 적절성
- [ ] 클릭베이트성 과장 없음

### Step 7: 플랫폼 최적화
- X: 280자 이내, 핵심 1포인트 + CTA (Hypebeast식 간결함)
- IG: 캡션 전체 구조 (오프닝-분석-클로저-CTA)
- IG 해시태그: 10~15개 (데패뉴식 카테고리 태깅 참조)

## 톤 가이드 (Hypebeast 적용)

### 사용할 동사
- "잇다 (bridge)", "재정의하다 (redefine)", "선도하다 (lead)"
- "해석하다 (interpret)", "큐레이션하다 (curate)", "제시하다 (present)"

### 피할 동사
- "대박이다", "미쳤다", "레전드", "역대급"
- "~인 것 같다", "~하게 된다"

### K-POP 아티스트 언급 톤
- Hypebeast식: "fashion ambassador", "style icon", "trendsetter"
- 팬덤 과몰입 금지, 산업적 관점 유지
- 아티스트의 패션 선택을 전략적 결정으로 분석

## 금지 사항
- 근거 없는 형용사 (화려한, 멋진, 대박)
- 아티스트 실명으로 얼굴 묘사
- 타 아티스트와의 비교
- "의" 남용
- 클릭베이트성 과장 표현
- 팬덤 슬랭 과다 사용
