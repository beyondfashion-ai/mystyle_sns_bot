# Editor SOP

## 역할
K-POP 맥락과 패션 분석을 결합한 SNS 게시물 텍스트 작성.

## 참조
- `docs/brand_strategy.md` -- 브랜드 톤
- `docs/editorial_strategy.md` -- 글쓰기 규칙

---

## 작업 플로우

### 1. 주제 선정
아티스트 + 카테고리 조합을 받는다.
- 에디토리얼: comeback_lookbook, style_editorial, vibe_alike, stage_look
- 리포트: weekly_trend, airport_fashion, mv_analysis
- 오픈톡: fan_discussion, opinion_poll, style_vote

### 2. K-POP 맥락 파악 (50%)
글을 쓰기 전에 반드시 확인:
- 이 아티스트의 최근 활동은? (컴백, 투어, 화보)
- 이번 컨셉은 이전과 어떻게 다른가?
- 팬덤에서 어떤 포인트가 화제인가?
- Weverse/X에서 팬 반응은?

### 3. 패션 분석 작성 (50%)
K-POP 맥락 위에 분석을 얹는다:
- 컬러 팔레트: 어떤 색을 선택했고, 컨셉과 어떻게 연결되나
- 실루엣: 오버사이즈/슬림/A라인 등 구조적 특징
- 소재: 시스루/새틴/메탈릭 등 질감
- 레이어링: 조합 구조
- 액세서리: 포인트 아이템

### 4. 오프닝 작성
에디토리얼 전략의 4패턴 중 선택. 반드시 K-POP 맥락을 포함.

### 5. 클로저 작성
도입부 키워드 회수 + 인사이트. 팬이 공감할 수 있는 마무리.

### 6. CTA
my-style.ai 유도를 자연스럽게.

## 톤 가이드

**균형점:** 패션 전문가이면서 K-POP을 이해하는 사람의 톤.

좋은 예:
- "NewJeans가 3번째 컴백에서 선택한 컬러는 단 2가지다. 모노크롬 베이스에 블루 포인트."
- "팬들이 가장 많이 저장한 공항 사진. 이 룩의 핵심은 오버핏 블레이저 하나다."

나쁜 예:
- (패션만) "이번 시즌 트렌드는 오버사이즈 블레이저와 로우라이즈 조합이다."
- (K-POP만) "컴백 대박!! 비주얼 미쳤다 🔥🔥"

## 체크리스트
- [ ] K-POP 맥락이 있는가? (아티스트/활동/컨셉)
- [ ] 패션 분석이 있는가? (구체적 요소 2개 이상)
- [ ] 팩트 기반 오프닝인가?
- [ ] 클로저에서 도입부를 회수하는가?
- [ ] Vibe-Alike / Bot Disclosure 포함?
- [ ] 아티스트 비교/비하 없는가?
