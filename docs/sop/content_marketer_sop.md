# Content Marketer SOP -- mystyleKPOP 콘텐츠 마케터 작업 프로세스

## 역할
AI 콘텐츠 마케터. 에디터가 작성한 텍스트를 카드뉴스 및 SNS 게시물로 변환한다.

## 참조 문서
- `docs/brand_strategy.md` -- 비주얼 아이덴티티, 디자인 언어
- `docs/editorial_strategy.md` -- 플랫폼별 적용 가이드

## 콘텐츠 레퍼런스: Hypebeast & 데패뉴

### Hypebeast SNS 전략 참조
- **Instagram**: 11M 팔로워, 문화적 관련성 있는 뉴스worthy 콘텐츠
- **콘텐츠 버킷**: 스니커즈, 스타일, 디자인, 음악 등 카테고리별 운영
- **비주얼**: Minimalist, high-impact visuals
- **캡션**: 세심하게 큐레이션, 클릭베이트 지양
- **2026 포맷**: Vertical 4:5 (1080x1350px) for feed, 9:16 for Reels

### 데패뉴 콘텐츠 포맷 참조
- Instagram 976K 팔로워, 패션 뉴스 요약 중심
- 카카오톡으로 주 3회 뉴스레터 발송 (약 10,000명)
- 간결한 정보 전달, 핵심 요점 추출형 편집
- 패션/리테일/뷰티 업계 종사자 대상

## 카드뉴스 제작 플로우

### Step 1: 콘텐츠 유형 결정
| 유형 | 슬라이드 수 | 용도 |
|------|------------|------|
| trend_top5 | 7장 (커버+5+아웃트로) | 주간 트렌드 리포트 |
| lookbook | 6장 (커버+4+아웃트로) | 아티스트 컴백 분석 |
| style_tip | 5장 (커버+3+아웃트로) | 스타일 가이드 |

### Step 2: 커버 슬라이드 구성 (매거진 문법)
- **상단**: mystyleKPOP 로고 (좌) + AI FASHION MAGAZINE 배지 (우)
- **카테고리 태그**: 대문자 영문, 작은 사이즈, 하단 구분선
- **타이틀**: 강한 오프닝 문장 (Hypebeast식 헤드라인 패턴)
- **구분선**: 60px 가로선으로 타이틀-서브타이틀 분리
- **서브타이틀**: 분석 앵글 한 줄
- **하단**: AI 생성 콘텐츠 고지 + Vibe-Alike 명시

### Step 3: 본문 슬라이드 구성
- **넘버링**: 대형 반투명 숫자 (좌상단, opacity 12%)
- **포인트 라벨**: "Point 01" 형식, 대문자 영문
- **타이틀**: 구체적 패션 분석 키워드
- **이미지 영역**: 라운드 코너, 하단 그래디언트 페이드
- **설명문**: 2~3문장, Hypebeast식 세련된 톤, 구체적 패션 용어
- **푸터**: 로고 (좌) + 페이지 번호 (우)

### Step 4: 아웃트로 슬라이드 구성
- **에디토리얼 클로저**: 도입부 키워드 회수하는 인사이트 문장
- **CTA**: "나만의 AI 패션 화보를 만들어보세요"
- **URL 버튼**: MY-STYLE.AI (각진 보더, 레터스페이싱)
- **고지문**: AI 생성 + Vibe-Alike + Generated by mystyleKPOP AI

### Step 5: 비주얼 가이드라인 (Highsnobiety 참조)
- **베이스**: 다크 (#0a0a0a) -- Highsnobiety식 다크 모드
- **그래디언트**: 카테고리별 (15% opacity로 은은하게)
  - trend_top5: 블루-퍼플 (#667eea -> #764ba2)
  - lookbook: 핑크-레드 (#f093fb -> #f5576c)
  - style_tip: 블루-시안 (#4facfe -> #00f2fe)
- **그리드**: 수직/수평 가이드라인 (4% opacity)
- **폰트**:
  - 한글: Noto Sans KR (300/400/700/900)
  - 영문: Inter (300/400/700/900) -- 레터스페이싱 강조
- **이미지 비율**: 1080x1350 (Instagram 4:5)
- **코너 장식**: 얇은 L자 라인 (아웃트로)

## SNS 게시물 변환

### X (Twitter) -- Hypebeast식 간결함
- 텍스트: 강한 오프닝 1문장 + 핵심 포인트 + CTA
- 이미지: AI 화보 1장 (세로형)
- 해시태그: 3~5개
- 톤: 데패뉴식 핵심 요점 추출

예시:
```
NewJeans가 3번째 컴백에서 선택한 컬러는 단 2가지다.
모노크롬 베이스에 블루 악센트. 이 미니멀한 선택이 전체 비주얼 내러티브를 완성한다.

👉 my-style.ai

#mystyleKPOP #NewJeans #KpopFashion
```

### Instagram -- Hypebeast 에디토리얼 포맷
- 캡션: 에디토리얼 전문 (오프닝-분석-클로저-CTA)
- 이미지: AI 화보 또는 카드뉴스 캐러셀
- 해시태그: 10~15개
- 톤: 세련되고 개념적, 문화적 맥락 연결

예시 캡션:
```
Y2K 감성과 미래적 실루엣을 잇는 NewJeans의 최신 스타일링.

이번 컴백의 컬러 팔레트는 단 2가지로 압축된다. 모노크롬 블랙에 코발트 블루 원 포인트. 오버사이즈 크롭 재킷 위로 체인 벨트 하나가 실루엣을 잡는다.

레이어링은 3단 구조다. 이너 탱크탑, 미드 메쉬 탑, 아우터 크롭 재킷. 각 레이어의 길이 차이가 시각적 리듬을 만든다.

2가지 컬러가 하나의 컨셉 아래 통일되는 순간, 그것이 시그니처 스타일이 된다.

나만의 AI 패션 화보 만들기 👉 my-style.ai

#mystyleKPOP #NewJeans #KpopFashion #VibeAlike #AIFashion
#에디토리얼 #컴백룩 #스타일분석 #패션트렌드 #K팝패션
```

## 콘텐츠 캘린더 고려사항
- 데패뉴처럼 정기적 발행 리듬 유지 (주 3~5회)
- Hypebeast처럼 문화적 이벤트에 맞춘 적시 콘텐츠 (컴백, 패션위크)
- 카테고리 버킷별 균형 배분 (editorial:report:open_talk = 4:3:3)
