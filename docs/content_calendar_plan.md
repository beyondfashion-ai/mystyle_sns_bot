# AI 패션 K-POP 매거진: SNS 콘텐츠 기획안 및 편성표

이 문서는 `mystyle_sns_bot`이 앞으로 자동으로 생성하고 게시할 **구체적인 콘텐츠 기획안과 주간 편성표**입니다. 
우리의 목표는 K-POP 팬덤을 사로잡고 최종적으로 `my-style.ai` 서비스로 유입시키는 것입니다.

---

## 1. 핵심 콘텐츠 포맷 (Core Content Formats)

우리 봇이 매일 생성할 3가지 핵심 에디토리얼(기사) 포맷입니다. 자동 스케줄러(10:00, 15:00, 20:00)에 맞춰 롤링됩니다.

### 포맷 A: [에디터의 컴백 예측 룩북] (Comeback Expectation)
*   **컨셉:** 곧 컴백하는 아이돌이나 현재 가장 핫한 아티스트의 다음 무대의상을 AI로 미리 입혀보는 기획.
*   **비주얼:** 아티스트 실명을 언급하되, 얼굴은 Vibe-Alike(분위기 미인) 가상 모델을 사용. 정면 화보 앵글.
*   **캡션 예시:** 
    > "곧 다가올 에스파(aespa)의 컴백! 🖤 STARSTYLE 매거진 에디터가 예측해 본 다음 무대의상은 바로 타격감 넘치는 '사이버펑크+테크웨어' 룩입니다. 이 스타일이 마음에 드신다면 여러분의 모델에게 직접 입혀보세요 👉 my-style.ai"
*   **프롬프트 가중치:** 외부 트렌드 크롤러에서 수집한 최신 `#무대의상` 관련 유행어(예: 메탈릭, 카고팬츠 등) 적극 반영.

### 포맷 B: [공항 패션 재해석] (Airport Style Reimagined)
*   **컨셉:** 최근 팬덤 사이에서 화제가 된 K-POP 아이돌의 사복/공항 패션을 AI의 시각으로 더 과감하게 리믹스.
*   **비주얼:** 자연스러운 스트릿 배경이나 공항 게이트 배경의 파파라치 혹은 스트릿 스냅샷 앵글.
*   **캡션 예시:**
    > "이번 주 화제의 공항패션을 AI로 한 층 더 힙하게 재해석해 보았습니다. ✈️ Y2K 무드를 듬뿍 담은 올 블랙 룩. 나만의 코디로 수정해보려면? 👉 my-style.ai/editor-pick"
*   **프롬프트 가중치:** 외부 트렌드 크롤러에서 `#공항패션`으로 검색된 키워드(예: 긱시크, 와이드팬츠, 데님) 반영.

### 포맷 C: [이주의 핫 트렌드 믹스] (Weekly Trend Mix)
*   **컨셉:** 순수하게 우리 봇 내에서 가장 반응이 좋았던(Internal Feedback) 아키타입과 스타일을 믹스매치한 '매거진 스페셜 요약 화보'.
*   **비주얼:** 매거진 표지(Cover) 느낌처럼 타이포그래피가 들어갈 수 있는 여백이 있는 화보 앵글.
*   **캡션 예시:**
    > "이번 주 STARSTYLE 독자들이 가장 열광한 키워드는 바로 `#발레코어`와 `청순 아키타입`의 만남! 🎀 트렌드를 앞서가는 이 룩을 내 손으로 직접 믹스매치해 보세요. 무료 체험 👉 my-style.ai"
*   **프롬프트 가중치:** 자체 `engagement_score` 분석기에서 뽑아온 최상위 효자 키워드 무조건 포함.

## 2. 주간 편성 달력 (Weekly Content Calendar)

매일 3회(10시, 15시, 20시) 스케줄러가 돌아갈 때, 요일별로 집중할 콘텐츠 테마입니다.

| 요일 | 오전 (10:00) | 오후 (15:00) | 저녁 (20:00) |
| :--- | :--- | :--- | :--- |
| **월요일** | **포맷 A** (새로운 한 주, 컴백 예측) | 포맷 B (주말 공항패션 리뷰) | 포맷 B (스트릿 스냅) |
| **화요일** | 포맷 B (데일리 사복 추천) | **포맷 A** (컴백 아이돌 컨셉 룩) | 포맷 C (인기 트렌드) |
| **수요일** | 포맷 C (주간 중간점검 인기 룩) | 포맷 B (공항패션 재해석) | **포맷 A** (무대의상 제안) |
| **목요일** | 포맷 B (스트릿 스냅) | 포맷 C (인기 트렌드) | **포맷 B** (불금 준비 룩) |
| **금요일** | **포맷 C** (이번 주 베스트 룩 정산) | **포맷 A** (주말 컴백팀 무대 예측) | 포맷 B (무대 의상 재해석) |
| **토/일** | 자유 프롬프트 (랜덤 매거진 생성) | **포맷 C** (주말의 매거진 커버) | 자유 프롬프트 (유저 믹스매치) |

---

## 3. 콘텐츠 생성 파이프라인 (작동 방식)

1. **[00:00] 자정 데이터 수집:** 일일 스케줄러가 돌아가면서 어제 성과가 좋았던 내부 트렌드(`analytics.js`)와 오늘 핫한 외부 K-POP 트렌드(`trendScraper.js`)를 긁어서 DB에 최신화합니다.
2. **[09:59] 텔레그램 초안 생성:** 스케줄러가 요일/시간에 맞는 포맷(A, B, C 중 하나)을 선택하고, DB에 저장된 "가중치 키워드"를 버무려서 텔레그램 관리자 방에 초안을 보냅니다.
3. **[10:00] 관리자 승인 (Human-in-the-loop):** 텔레그램에서 "✅ 승인"을 누르거나 "✏️ 수정"을 통해 텍스트를 살짝 다듬습니다. (법적 리스크 방지용 Vibe-Alike 준수 확인)
4. **승인 직후:** X(Twitter)와 인스타그램에 통합 포스팅 로직(`bot.js`)을 타고 깔끔한 에디토리얼 형식으로 동시 발행됩니다. 
5. 다음날 자정에 이 게시물이 얼마나 호응을 얻었는지 다시 수집하여 1번으로 돌아옵니다. (영원한 셀프 발전 루프)
